1. Instalacja git na maszynie za pomocą polecenia "sudo apt-get install git" (zadanie pominięte, git już był zainstalowany).

2. Sklonowanie repozytorium zdalnego za pomocą komendy "git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021.git" oraz utworzenie grafu obrazującego relacje pomiędzy branchem lokalnym oraz zdalnym.

3. Zapoznanie się z instrukcjami laboratoriów "MIFT2021/Lab instructions/Lab 01 - Git.txt".

4. Utworzenie nowego brancha oraz automatyczne przełączenie się na niego za pomocą komendy "git checkout -b RN306479", utworzenie folderu "RN306479/Lab01", w folderze "Lab01" utworzenie pliku "spawozdanie.txt", utworzenie nowego commita za pomocą komendy "git add ." (to mało profesjonalny sposób, powinienem użyć "git add nazwa_pliku" żeby nie doszło do dodania niechcianych plików, w tym przypadku znikoma ilość zmian sprawia że można użyć takiego uproszczenia) oraz "git commit -am "RN306479 create basic catalog", dodanie grafu do repozytorium oraz wykonanie commita "git commit -am "RN306479 add graph to repo".

5. Opisanie dotychczasowych działań w pliku "sprawozdanie.txt".

6. Dodanie zrzutów ekranu do folderu Lab01, wykonanie commita "git add ." oraz "git commit -am "RN306479 add screenshots", zmiana ich nazw do odpowiadających im zadań.

7. Merge branch "RN306479" do brancha "master", "git checkout master" - przełączenie na branch master, "git pull" - ściągnięcie zmian z repozytorium GitHub, "git merge RN306479" - zmergowanie mojego brancha z branchem "master", "git push" - wypchnięcie do zdalnego repozytorium.
