1. Uruchomienie maszyny wirtualnej Linux Ubuntu

2. Pobranie gita za pomoca komendy: sudo apt install git

3. Pobranie repozytorium za pomoca komendy: git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021.git

4. Otworzenie instrukcji do danych zajec, zapoznanie sie z nimi

5. Stworzenie nowego brancha o nazwie JF306439 za pomoca komendy: git branch -b JF306439

6. Stworzenie w nowym branchu folderu o nazwie JF306439 oraz podfolderu Lab01

7. Stworzenie pliku sprawozdanie.txt, opisanie w nim wykonanych czynnosci

8. Dodanie pliku do repozytorium gita za pomoca komend: git add JF306439	oraz	git commit -m "dodanie sprawozdania"

9. Wyslanie plikow na GitHub za pomoca komendy: git push

10. Dodanie zrzutow ekranu do repozytorium poprzez: add, commit oraz push

11. Przejsce na branch master poprzez: git checkout master

12.  Wykonanie merge wlasnego brancha do branch master za pomoca komendy: git merge JF306439	oraz	git push
