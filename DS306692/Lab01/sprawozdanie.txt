1. Przed zajeciami uzylem komend do instalacji gita
	sudo dnf update-y
  	sudo dnf install git -y
2. Uzylem polecenia
	git clone
   do sklonowania repozytorium
   Utworzylem graf
3.Zapoznalem sie z instrukcją
4.Utworzylem nowy branch instrukcja
   	git checkout -b DS306692
  Stworzyłem odpowiednie katalogi
   	mkdir 
  Stworzylem plik   
   	touch sprawozdanie.txt
  Dodalem plik do repozytorium
   	git add sprawozdanie.txt
   	git commit -m "opis"
  Tak samo uczynilem dla pliku z grafem

5.Napisalem sprawozdanie..

6.Umiescilem zrzuty ekranu w odpowiednim katalogu a takze dodalem je do repozytorium wykonuje commita i push 

7.Przejscie na branch master przy uzyciu 
	git checkout master
  Merge:
	git merge DS306692
	
  Push na remote:
	git pull (git wymagał)
	git push
