3. Stworzenie brancha

    git checkout MS306511

4. Dodawanie brancha:
    git add --all - dodanie wszystkich nowo dodanych plikow do staging area
    git commit -m "komentarz" -wprowadzenie zmian ze stagging area do repozytorium, zastosowanie opcji -m pozwala na wprowadzenie komentarza
    git push - wprowadzenie zmian do repozytorium zdalnego

5. merge
Zlaczenie mojego lokalnego brancha z branchem master, spushowanie wszystkiego na mastera
