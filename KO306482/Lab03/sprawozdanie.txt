zadanie 5
  Stworzenie folderu do pracy
	* mkdir <nazwa folderu>
  Utworzenie pliku o nazwie ,,DockerFile"
	* nano Dockerfile
  Konfiguracja pliku konfiguracyjnego obrazu kontenera
	w pliku DockerFile:

	FROM ubuntu:latest
	RUN apt-get -y update
	RUN apt-get -y install git

  Stworzenie obrazu kontenera
	* docker build -t 'git-image' .

zadanie 6
  Uruchomienie kontenera
  	* docker run -t git-image bash
  Sklonowanie repozytorium
  	* git clone <adres repozytorium>
