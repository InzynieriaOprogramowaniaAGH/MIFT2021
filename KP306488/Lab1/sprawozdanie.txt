1) Pierwszym zadaniem było zainstalowanie klienta git, wykonałem to polecenie za pomocą polecenia: sudo apt-get install git
2) Następnie za pomocą polecenia git clone zklonowałem repozytorium: git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021.git
3) Następnie stworzyłem nowy folder o nazwie KP306488 oraz nowy branch za pomocą poleceń:
-git branch KP306488
-git checkout KP306488
-mkdir Lab1
-git add sprawozdanie.txt
-git commit -m "Pierwszy commit"
4) Następnie dodałem graf:
-git add graph.png 
-git commit -m "Dodanie grafu"
5) Zamieszczenie zrzutów ekranu
