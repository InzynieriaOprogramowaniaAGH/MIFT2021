1. klonowanie repo
git clone <url do repo>

2. tworzenie brancha
git branch <nazwa brancha>

3. przełączenie sie na brancha
git checkout <nazwa brancha>

4. tworzenie folderów
mkdir <nazwa folderu>

5. tworzenie pliku
echo "" > plik.txt

6. dodawanie zmian do commita
git add <nazwa pliku albo folderu>
ja dawałem git add EK306459 żeby dodać zmiany z całego folderu

7. commitowanie
git commit

8. pushowanie 
git push

9. mergowanie brancha
przejście na master -> git merge <nazwa brancha>
