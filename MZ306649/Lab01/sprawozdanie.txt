Wykorzystywanie polecenia

git clone - polecenie klonuje istniejące repozytroium do wybranego katalogu

git branch <nazwa> - polecenie tworzy nowy branch o podanej nazwie

git checkout <nazwa brancha> - polecenie przłącza na podany branch

mkdir <nazwa> - utworzenie katalogu o podanej nazwie

touch <nazwa> - stworzenie pliku o podanej nazwie

git add <nazwa> - dodanie do repozytorium pliku o podanej nazwie

git commit - polecenie zapisuje zmiany w repozytorium (-m "komentarz commita")

git merge <nazwa brancha> - polecenie łączy historię podanego brancha do brancha na którym użytkownik wykonuje polecenie

git push - polecenie przesyła pliki z lokalnego repozytorum do repozytorium zdalnego
