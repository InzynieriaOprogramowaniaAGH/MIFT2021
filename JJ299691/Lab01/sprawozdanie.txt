1. Uruchomienie gitbash

2. Pobranie repozytorium za pomoca komendy: git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021.git

3. Otworzenie instrukcji do danych zajec, zapoznanie sie z nimi

4. Stworzenie nowego brancha o nazwie JJ299691 za pomoca komendy: git branch -b JJ299691

5. Stworzenie w nowym branchu folderu o nazwie JJ299691 oraz podfolderu Lab01

6. Stworzenie pliku sprawozdanie.txt, opisanie w nim wykonanych czynnosci

7. Dodanie pliku do repozytorium gita za pomoca komend: git add oraz git commit 

8. Wyslanie plikow na GitHub za pomoca komendy: git push

9. Dodanie zrzutow ekranu do repozytorium poprzez: add, commit oraz push

10. Przejsce na branch master poprzez: git checkout master

11.  Wykonanie merge wlasnego brancha do branch master za pomoca komendy: git merge JJ299691 oraz git push
