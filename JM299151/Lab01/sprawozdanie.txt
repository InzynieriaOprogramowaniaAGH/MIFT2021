Temat zajęć: Wprowadzenie. Konfiguracja repozytorium GIT. Środowisko GIT i jego podstawowa obsługa.

Zadanie:
1) Zainstalowanie klienta git.
sudo apt-get install git

2) Sklonowanie repozytorium oraz narysowanie grafu zależności między repozytoriami,
git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021.git

3) Zapoznanie się z instrukcją dotyczącą zajęć.

4) Stworzenie nowego branch'a w repozytorium lokalnym, a następnie przełączenie się na niego
git checkout -b JM299151
Stworzenie nowego folderu Lab01
mkdir Lab01 
Stworzenie pliku ze sprawozdaniem
touch sprawozdanie.txt
Dodanie do folderu plików: sprawozdanie.txt oraz graf.png do folderu LAB01

5) Wykonanie zrzutów ekranu wykonanych operacji oraz dodanie ich do repozytorium 
git add .
git commit -m "//komentarz"

6) Wykonanie mere utworzonego brancha do branch master.main. 
git checkout master
git merge JM299151
git pull
git push
