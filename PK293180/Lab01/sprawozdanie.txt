1. Zainstalowno klienta git przy pomocy sudo apt-get install git

2. Wykorzystujac git clone sklonowano repozytorium. Nastepie narysowano graf zaleznosci. 

3. Zapoznano sie ze sprawozdaniem korzystajac z vim

4. Przy pomocy checkout -b stworzono nowy branch oraz autoamtycznie przelaczono sie na niego. Nazwa brancha PK293180, w ktorym stworzono folder o tej samej nazwie przy pomocy mkdir, pozniej w tym folderze przy pomocy tej samej komendy stworzono Lab01. Plik sprawozdanie.txt zostal stworzony przy pomocy touch.

5. Wykonano commita oraz push do poprzednich zadan przy pomocy git add * git commit -m "<tresc commita>"

6. Wykonano commit oraz push przy pomocy polecen tych samych co wyzej, do tego push przy pomocy git push

7. Przelaczono sie na branch master przy pomocy git checkout master, nastepnie przy pomocy pull sciagnieto zmiany, nastepnie wykonano git push i na koncu wykonano git merge PK293180
