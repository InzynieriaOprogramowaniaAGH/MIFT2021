zad 1. Program wymagany w poleceniu miałem już zainstalowany.

zad 2. Wykonałem clonowanie repozytorium za pomocą polecenia:
	git clone <adres repozytorium>
	Dodałem graf zależności.

zad 3. Przegląd instrukcji w katalogu <repo>/Lab instructions/Lab 01 - Git.txt"

zad 4. 	Stworzenie nowego brancha i przełączenie na nią za pomocą polecenia
	git checkout -b MZ306526
	Polecenie tworzy nam nowego brancha o nazwie MZ306526 i przełącza na niego.
	Stworzenie odpowiednich katalogów oraz dodanie do nich plik sprawozdanie.txt
	Wykonałem polecenie git add <nazwa katalogu> a następnie wykonałem commit poleceniem git commit - m 'lab4'

zad 5. 	Modyfikacja pliku sprawozdanie.txt o opisy poszczególnych zadań,
	wykonałem nowego commita poleceniem git commit -m 'message'.

zad 6. Dodanie zrzutów ekranu, wykonanie kolejnych commitów i wykonanie pushu na swojego brancha.
	Zmaiana nazw plików i powtórzenie czynności.

zad 7.Wykonanie merge naszego brancha na master za pomocą polecenia 
	git merge
	następnie wykoannie git pull oraz git push
