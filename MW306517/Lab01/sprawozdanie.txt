Zadanie 1. Klient Git został zainstalowany wcześniej razemz z GUI przy pomocy pliku exe pobranego z internetu.

Zadanie 2. Repozytorium zdalne sklonowano wykorzystując GUI porzez podanie adresu źródłowego oraz ścieżki docelowej w systemie, 
co było równoznaczne z wykonaniem polecenia 'git clone'. Wykorzystując Program Microsoft Paint wykonano graf przedstawiający repozytorium lokalne oraz zdalne.

Zadanie 3. Zapoznanie się z instrukcją dotyczącą laboratorium

Zadanie 4. Utworzenie oraz równoczesne przełączenie się na nowego brancha w repozytorium lokalnym o nazwie "MW306517". 
Następnie utworzono folder o takiej samej nazwie oraz podfolder Lab1. Wykorzystane polecenia wykonano przy pomocy konsoli Git Bash:
'git checkout -b MW306517'
'mkdir MW306517'
'mkdir Lab1'
W tak przygotowanych folderach utworzono plik sprawozdanie.txt, który następnie dodano do repozytorium poleceniem 'git add sprawozdanie.txt'. 
Wykonano commita poleceniem 'git commit -m "komentarz"', po czym dodano jeszcze graf z polecenia 2 i również wykonano commita.

Zadanie 5. W przygotowanym pliku tekstowym umieszczono krótki opis wykonywanych zadań, po czym komendą 'git push origin MW306517' umieszczono dytchczas wykonaną pracę 
na repozytorium zdalnym upewniając się przy tym, że znajdujemy się na odpowienim branchu poleceniem 'git checkout MW306517'.

Zadanie 6. W folderze Lab1 umieszczono wszystkie screeny przedstawiające wykonanie zadań z instrukcji 
oraz tak samo jak poprzednio utworzono commita i umieszczono zmiany na zndalnym repozytorium.
'git add .'
'git commit -m "komentarz"'
'git push origin MW306517'
