	* ssh-keygen -t ed25519 -C "juliaszandula@outlook.com"
	Dodałam klucz SSH do agenta ssh poleceniem:
	* ssh-add ~/.ssh/id_ed25519
	Zarejestrowałam klucz po stronie serwera za pośrednictwem strony github.com.
	Poinformowałam git'a o zmianie sposobu logowania wykorzystując następujące komendy:
	* git remote -v
	* git remote set-url origin git@github.com:InzynieriaOprogramowaniaAGH/MIFT2021.git
	* git remote -v
