1. Klient git był zainstalowany na moim stanowisku

2. Klonowanie repozytorium wykonałem poleceniem w terminalu :
"git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021.git"

Narysowano prostą grafikę pokazującą związek między repozytorium lokalnym a githubowym. (zadanie2.1, zadanie2.2)

3. Otwarto plik z poleceniami do wykonania: MITF2021/lab01-Git.txt (zadanie3)

4. Utworzenie brancha: "git checkout -b MZ306522"

5. Dodanie nowego folderu lokalnie z poleceniami: "mkdir Lab01", następnie dodano plik sprawozdanie.txt: "touch sprawozdanie.txt"
następnie wykonano "git add ." "git commit -a" oraz "git push --set-upstream origin MZ306522"

6. Utworzono odpowiednie foldery i dodano zrzuty ekranu dla zadań 

7. Wykonanie merge brancha (MZ306522) do master:
- "git checkout master" 
- "git merge MZ306522"
- "git pull"
- "git push"
