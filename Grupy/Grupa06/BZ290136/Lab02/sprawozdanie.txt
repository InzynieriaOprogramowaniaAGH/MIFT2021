1. Wygenerowano klucz SSH:
ssh-keygen -t ed25519 -C "bartlomiejzieja@gmail.com"

2. Uruchomiono agenta SSH:
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

3. Dodano klucz do githuba, kopiujac go poleceniem:
xclip -selection clipboard < ~/.ssh/id_ed25519.pub

4. Utworzono tag z wiadomoscia i spushowana:
git tag -a BZ290136_Lab02 -m "Bartlomiej Zieja, everything is ok"
git push origin BZ290136
git push -u --follow-tags
