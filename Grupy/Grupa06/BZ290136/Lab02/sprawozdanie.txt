Najpierw przelaczono sie na swojego brancha:
git checkout BZ290136 --

1. Wygenerowano klucz SSH:
ssh-keygen -t ed25519 -C "bartlomiejzieja@gmail.com"

2. Uruchomiono agenta SSH:
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

3. Dodano klucz do githuba, kopiujac go poleceniem:
xclip -selection clipboard < ~/.ssh/id_ed25519.pub

4. Utworzono tag z wiadomoscia i spushowana:
git tag -a BZ290136_Lab02 -m "Bartlomiej Zieja, everything is ok"
git push origin BZ290136
git push -u --follow-tags

5. Stworzono zadana konfiguracje folderow:
mkdir -p Grupy/Grupa06
mv -v BZ290136 Grupy/Grupa06/
mkdir Grupy/Grupa06/BZ290136/Lab02

6. Uporzadkowano sturkture branchy poprzez utowrzenie prywatnego brancha:
git checkout -b Grupa06-BZ290136_Lab02

7. Napisano githooka sprawdzajacego komentarze do dodawanych commitow. Edytowano plik commit-msg.sample w folderze .git/hooks i zmieniono jego nazwe na commit-msg. Tresc skryptu:

stringToCheck='BZ290136*'
if ! grep "$stringToCheck" "$1"; then
	echo "Commit message has wrong start - commit is not done"
	exit 1;
fi
	echo "Commit message has correct start - commit is done"
	exit 0;

8. Wykonano pusha z brancha lokalnego do grupowego i zamknieto branch osobisty:
git checkout Grupa06
git merge Grupa06-BZ290136_Lab02  -m "BZ290136 lab02 zad 8" 
git push https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021/pulls Grupa06:master
git request-pull BZ290136_Lab02 https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021/pulls Grupa06:master
