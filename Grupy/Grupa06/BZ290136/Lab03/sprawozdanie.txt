1. Utwrzono brancha Grupa06-BZ290136_Lab03
2. Skopiowano do odpowiedniego folderu utworzony na poprzednich zajec skrypt git hooka:
cp commit-msg ../../Grupy/Grupa06/BZ290136/Lab03/
3. Zaisntalowano docker:
	sudo apt-get remove docker docker-engine docker.io containerd runc
	sudo apt-get update
	
	sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg \
    lsb-release
    
	curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

	echo \
  "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
  
  sudo apt-get update
  
4. Utoworzono konto na dockerhub.

5. Utworzono plik Dockerfile:
	touch Dockerfile
	
	z nastpujaca zawartoscia:
	
	FROM ubuntu:latest
	RUN apt-get update
	RUN apt-get install -y git
	
	i dodano go do swojego brancha.
	
6. Uruchomienie kontenera i sciagniecie na niego naszego repo:
	sudo docker build -t myimage .	<- zbudowanie obrazu
	sudo docker run -it myimage		<- uruchomienie obrazu
	git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021 <- sklonowanie repo
	git branch -a				<- sprawdzenie czy znajduje sie tam moj branch
