	ssh-keygen -t ed25519 -C "m.tomaszewski11@interia.pl"
1.2. Uruchomiem ssh-agent:
	eval 'ssh-agent -s'
1.3. Dodaem klucz do agenta:
	ssh-add ~/.ssh/id_ed25519
1.4. Przypisaem klucz do konta i polaczylem sie z repo

2.1  Dodalem commit z tagiem:
	git tag -a MT299769_Lab02 -m "komentarz"
2.2  Wrzuciłem tą zmianę na swojego brancha na origin, a następnie remote/origin używając: 
	git push --follow-tags

3.1  Uporządkowałem foldery i stowrzyłem brancha Grupa06-MT299769-Lab02
4.1  Napisałem git hooka:
	word="MT299769"
	isPresent=$(grep ^"$word" $1)

	if [[ -z $isPresent ]]
  	then echo "commit nie rozpoczyna się od poprawnej frazy"; exit 1;
  	else echo "commit ok"; exit 0;
	fi
