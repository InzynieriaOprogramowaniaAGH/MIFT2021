1. Zainstalowanie Kubernetesa:
	curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
	sudo install minikube-linux-amd64 /usr/local/bin/minikube
	minikube start
	minikube kubectl -- get po -A
	minikube dashboard
2.Minikube dashboard pozwala nam na:
a)przegladanie aplikacji uruchomionych na klastrze
b)troubleshoting
c)tworzenie i zarzadzanie klastrem
3.Uruchomienie komunikatora w Kubernetes
	alias kubectl="minikube kubectl --"
	kubectl run komunikator --image=gzelek8/lab05-gzelek --port=9999 --labels app=komunikator
4.Przekierowanie portu:
	kubectl port-forward komunikator 9999:3000
