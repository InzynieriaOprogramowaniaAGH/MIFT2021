Laboratorium nr.5
1. Stworzyłem brancha na nowe zajęcia
	git branch Grupa08-MO306686_Lab05

2. Zapoznałem się oprogramowaniem Jenkins oraz z diagramem przedstawionym na zajęciach

3.1 Rozpocząłem tworzenie kontenera Dockerowego z Jenkinsem
	docker network create jenkins
	
	docker run \
  	--name jenkins-docker \
  	--rm \
  	--detach \
  	--privileged \
  	--network jenkins \
  	--network-alias docker \
  	--env DOCKER_TLS_CERTDIR=/certs \
  	--volume jenkins-docker-certs:/certs/client \
  	--volume jenkins-data:/var/jenkins_home \
  	--publish 2376:2376 \
  	docker:dind \
  	--storage-driver overlay2
3.2 Stworzyłem Dockerfile uwzględniający Jenkinsa w kontenerze
	touch Dockerfilejenkins
	
3.3 Zbudowałem obraz Dockerowy
	docker build -t myjenkins-blueocean:1.1 . -f Dockerfilejenkins
3.4 Uruchomiłem  obraz Jenkinsa w kontenerze Dockerowym
	docker run \
  	--name jenkins-blueocean \
  	--rm \
  	--detach \
  	--network jenkins \
 	--env DOCKER_HOST=tcp://docker:2376 \
 	--env DOCKER_CERT_PATH=/certs/client \
 	--env DOCKER_TLS_VERIFY=1 \
 	--publish 8080:8080 \
  	--publish 50000:50000 \
 	--volume jenkins-data:/var/jenkins_home \
  	-volume jenkins-docker-certs:/certs/client:ro \
  	myjenkins-blueocean:1.1 
3.5 Po zalogowaniu do Jenkinsa stworzyłem Joba, który pobiera repozytorium i uruchamia docker-compose
