1. Stworzenie kontenera dockerowego z Jenkinsem za pomocÄ…:
	docker network create jenkins
	docker run \
  		--name jenkins-docker \
  		--rm \
  		--detach \
  		--privileged \
  		--network jenkins \
  		--network-alias docker \
  		--env DOCKER_TLS_CERTDIR=/certs \
  		--volume jenkins-docker-certs:/certs/client \
  		--volume jenkins-data:/var/jenkins_home \
  		--publish 2376:2376 \
  		docker:dind \
  		--storage-driver overlay2
  		
2. Stworzenie pliku Dockerfile

3. Zbudowanie obrazu dockera: $ sudo docker build -t myjenkins-blueocean:1.1 .

4. Uruchomienie jenkinsa i instalacja

5. Utworzenie repo na dockerhubie i wrzucenie obrazu Jenkinsa

	$ sudo docker push new-repo:tagname
