Wykorzystane polecenia: 

ssh-keygen -t ed25519 -C "your_email@example.com" - wygenerowanie nowego klucza ssh

ssh-add ~/.ssh/id_ed25519 - dodanie ssh

clip < ~/.ssh/id_ed25519.pub - skopiowanie klucza ssh do schowka

git remote -v - wyświetlenie informacji o sposobie połączenia
