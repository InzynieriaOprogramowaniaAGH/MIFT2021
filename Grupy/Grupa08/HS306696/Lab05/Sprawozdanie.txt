1. Zapoznałem się z diagramem omawianym na zajęciach i oprogramowaniem Jenkins.
2. Stworzyłem kontener dockerowy z Jenkinsem według instrukcji:
	a)
	$ docker network create jenkins
	$ docker run \
  		--name jenkins-docker \
  		--rm \
  		--detach \
  		--privileged \
  		--network jenkins \
  		--network-alias docker \
  		--env DOCKER_TLS_CERTDIR=/certs \
  		--volume jenkins-docker-certs:/certs/client \
  		--volume jenkins-data:/var/jenkins_home \
  		--publish 2376:2376 \
  		docker:dind \
  		--storage-driver overlay2
 	b) utworzyłem odpowiedni plik Dockerfile
 	c) zbudowałem obraz dockera: $ docker build -t myjenkins-blueocean:1.1 .
 	d) $ docker run \
		  --name jenkins-blueocean \
		  --rm \
		  --detach \
		  --network jenkins \
		  --env DOCKER_HOST=tcp://docker:2376 \
		  --env DOCKER_CERT_PATH=/certs/client \
		  --env DOCKER_TLS_VERIFY=1 \
		  --publish 8080:8080 \
		  --publish 50000:50000 \
		  --volume jenkins-data:/var/jenkins_home \
		  --volume jenkins-docker-certs:/certs/client:ro \
		  myjenkins-blueocean:1.1
	e) zakończyłem instalację Jenkinsa według instrukcji ze strony
3. Utworzyłem nowe repozytorium na DockerHubie i wrzuciłem na nie obraz swojego Jenkinsa
4. Uruchomiłem docker-compose z poziomu Jenkinsa
