następnie należy dodać klucz ssh do gita (na stronie)
polecenie: clip < ~/.ssh/id_ed25519.pub -> kopiuje ssh do schowka następnie należy go wkleić w odpowiednim miejscu na stronie

wyświetlenie połączenia (powinno być po https):
$ git remote -v

następnie należy zmienić połączenie SSH na url origin git@github.com:InzynieriaOprogramowaniaAGH/MIFT2021.git:
poleceniem: git remote set-url origin git@github.com:InzynieriaOprogramowaniaAGH/MIFT2021.git

oraz ponowne sprawdzenie poprzez: $ git remote -v

nastepnie robie commita na branczu

$ git tag -a WL306675_Lab02 -m "Wiktor Ligeza, Lab 02 wszystko działa chyba"


następnie po stworzeniu taga pushuje na origina 
$ git push --follow-tags
