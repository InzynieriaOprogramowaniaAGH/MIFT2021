1. Wygenerowałem klucz ssh za pomocą komendy: ssh-keygen -t ed25519 -C "moj_adres_email".
   Wpisalem haslo. 
   Dodalem klucz do agenta ssh: eval 'ssh-agent -s'.
   Dodalem ssh: ssh-add ~/.ssh/id_ed25519.
   Zarejestrowalem klucz po stronie serwera.
   Zmienilem polaczenie na ssh za pomoca komendy: git remote set-url origin git@github.com:InzynieriaObliczeniowaAGH/MIFT2021.git

2. Nadalem tag za pomoca: git tag -a MK299627_Lab02 -m "wszystko dziala"
   Oraz git push --follow-tags

3. Uporzadkowalem branche wedlug instrukcji poprzez utworzenie nowego brancha z brancha Grupa07:
   git checkout -b Grupa07-MK299627_Lab02
   Wykonalem na nim git merge Grupa07.
   Po kazdych wiekszych zmianach na tymczasowym branchu wykonywalem git commit, nastepnie zmienialem brancha na Grupa07 i wykonywalem merge z branchem tymczasowym.

4. Napisalem git hooke'a poprzez otwarcie folderu .git/hooks, edycje pliku commit-msg.sample, oraz usuniecie rozszerzenia .sample, aby hooke byl widoczny.
   Napisany przeze mnie kod znajduje sie na zrzucie ekranu zad_4.

5. Wykonalem merge z brancha grupowego z branchem tymczasowym, w celu zapisania zmian.
   Usunalem tymczasowy branch osobisty za pomocą: git branch -d Grupa07-MK299627_Lab02.
