1. Tworze brancha na czas zajec:
	git checkout -b Grupa07-DK306669_Lab03
	
2. Dodaje hook file z poprzednich zajec do mojego folderu z lab3

3. Zainstalowalem dockera na systemie ubuntu
	sudo apt-get update

 	sudo apt-get install \
    		apt-transport-https \
    		ca-certificates \
    		curl \
    		gnupg \
    		lsb-release	
    		
    	curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo 		gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
    	
    	echo \
  	"deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive 	keyring.gpg] https://download.docker.com/linux/ubuntu \
  	$(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
  	
  	sudo apt-get update

 	sudo apt-get install docker-ce docker-ce-cli containerd.io
 	
4. Stworzylem konto w dockerhub.

5. Utworzylem plik Dockerfile, ktory ma bazowac na najnowszym ubuntu.

6. Skonfigurowalem kontener
	
	sudo docker build -t ms_docker .
	sudo docker run -it (tutaj odpowiednia wartosc)
	git clone (repo)
