1. Przełączyłem się na branch z poprzednich zajęć i opublikowałem obraz w Docker Hub
docker images
docker tag
docker push


2. Wybrałem komunikator node chat https://github.com/binhxn/node-chat-app

3. Stworzyłem plik Dockerfile i odpowiednio go skonfigurowałem 
 docker build -t build_agent .

4. Stworzyłem nowy Dockerfile odpowiedzialny za testowanie komunikatora w test/Dockerfile
docker build -t test_agent test

5. Zainstalowałem docker-compose
sudo curl -L "https://github.com/docker/compose/releases/download/1.29.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose; Sprawdzoono poprawność instalacji: docker-compose --version

Utworzyłem plik docker-compose.yml 
Uruchomiłem odpowiednio przygotowaną konfigurację za pomocą docker-compose up
