Stworzenie kontenera dockerowego z Jenkinsem 
- sudo docker network create jenkins
- sudo docker run --name jenkins-docker --rm --detach \
  --privileged --network jenkins --network-alias docker \
  --env DOCKER_TLS_CERTDIR=/certs \
  --volume jenkins-docker-certs:/certs/client \
  --volume jenkins-data:/var/jenkins_home \
  --publish 2376:2376 docker:dind --storage-driver overlay2
- utworzenie Dockerfile
- sudo docker build -t myjenkins-blueocean:1.1 .
Nie dało się wysłać obrazu na dockerhub, więc zmieniłem tag na bhajduk/myjenkins-blueocean:1.1
- sudo docker push bhajduk/myjenkins-blueocean:1.1
Przez localhost:8080 nie uzyskałem dostępu do strony startowej Jenkinsa. Udało mi się poprzez adres ip kontenera. 
Po uruchomieniu kontenera z jenkinsem i przejściu pod wskazany adres utworzono użytkownika, a następnie dodano nowego joba. 
cd Grupy/Grupa02/BH306443/Lab04
curl -L "https://github.com/docker/compose/releases/download/1.29.1/docker-compose-$(uname -s)-$(uname -m)" -o /var/jenkins_home/docker-compose

eval "$(docker-machine env default)"
/var/jenkins_home/docker-compose up
