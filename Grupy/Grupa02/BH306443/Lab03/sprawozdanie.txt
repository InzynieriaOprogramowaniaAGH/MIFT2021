Instalacja dockera poprzez komendę sudo apt-get install docker.io
Start dockera: 
sudo systemctl start docker
sudo systemctl enable docker
Utworzony plik Dockerfile:
FROM ubuntu:latest
RUN apt-get update && apt-get upgrade -y
RUN apt-get install git -y
Uruchomienie obrazu z pliku Dockerfile - sudo docker built . -t ubuntu 
Uruchomienie kontenera z sesją bash - sudo docker run -it ubuntu bash
