instalacja gita - sudo apt-get install git
klonowanie repozytorium - git clone [linkDoGithuba]
tworzenie brancha - git branch [nazwaBrancha]
"chodzenie" po branchach - git checkout [nazwaBrancha]
tworzenie pliku - touch sprawozdanie.txt
dodanie plikow do staging area - git add [plik] lub git add .
tworzenie commita - git commit -m "commit message"
wysylanie zmian do remotea - git push [origin] [branch]
merge z galezi master - git merge JK306462
usuwanie galezi - git branch -d jk306462
