Lab05 Temat: Jenkins jako CI/CD driver - w kontenerze.

0. Zapoznałam się z diagramem omawianym na zajęciach.

1. Zapoznałam się z oprogramowaniem Jenkins: https://www.jenkins.io/.

2. Stworzyłam kontener dockerowy z Jenkinsem posługując się instrukcją https://www.jenkins.io/doc/book/installing/docker/
w ramach tego zadania wykonałam:
    - utworzyłam sieć: docker network create jenkins
    - pobralam i uruchomilam kontener docker:dind (aby móc wykonywac polecenia w dockera w wezlach jenkinsa)
    - utworzylam dockerfile i zbudowalam obraz
    - uruchomilam kontener ze zbudowanego obrazu (dzieki czemu mozna uruchomic jenkinsa)
    -uruchomilam jenkinsa w przegladarce
    -znalazlam ID kontenera poleceniem: docker ps
    -aby znalezc haslo administratorskie uzylam komendy: docker logs ID-kontenera
    -zalogowałam sie i zainstalowałam potrzebne pluginy
    -utworzylam konto i zmienilam URL na localhost:8080/jenkins

3. Wrzuciłam obraz swojego Jenkinsa na DockerHub'a z odpowiednim tagiem.
docker tag nazwa-obrazu:tag login-dockerhub/nazwa-obrazu:tag
docker login
docker push login-dockerhub/nazwa-obrazu:tag
