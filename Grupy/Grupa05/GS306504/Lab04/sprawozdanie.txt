1. 
- Stworzono nowy branch na te zajecia:

git checkout -b Grupa05-GS306504_Lab04

- Zalogowano do dockera:

docker login -u gregwell

- zbudowano obra dockerowy

docker build -t gregwell/grupa05gs306504lab03:latest .

- zpushowano 

docker push gregwell/grupa05gs306504lab03

2. Znaleziono komunikator na githubie: https://github.com/binhxn/node-chat-app


3. Napisano Dockerfile-build:

FROM node:latest

WORKDIR /data

RUN git clone https://github.com/binhxn/node-chat-app.git /data/app

WORKDIR /data/app

RUN npm install

EXPOSE 3000

CMD ["npm", "start"]


4. Napisano Dockerfile-test

FROM nodechat-build:latest

WORKDIR .

CMD [ "npm", "test" ]

5. Napisano docker-compose.yml (https://docs.docker.com/compose/)

version: '3.3'
services: 
    build:
        container_name: nodechat-build
        image: nodechat-build:latest
        ports:
            - "3000:3000"
        build: 
            context: .
            dockerfile: Dockerfile-build
    test: 
        container_name: nodechat-test
        image: nodechat-test:latest
        build: 
            context: .
            dockerfile: Dockerfile-test
        depends_on:
        - build

Stworzono środowisko które ściąga z repo najnowszą wersję komunikatora, buduje go i testuje czy działa.


Do punktów 2-5 brak zrzutu ekranu ponieważ pliki znajdują się w katalogu, a punkt 2 to tylko znalezienie aplikacji.
Zrzuty do ostatniego zadania pokazują poprawne odpalenie kontenerów

Polecenie do odpalania docker-compose to:

docker-compose up

Użyto również

docker-compose up --build
