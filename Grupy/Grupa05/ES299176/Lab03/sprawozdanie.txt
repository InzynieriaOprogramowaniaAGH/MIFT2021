Zadanie 5. 
Utworzyłam plik Dockerfile, który bazuje na najnowszym ubuntu, ma doinstalowanego gita i wykonałam klonowanie repozytorium:

FROM ubuntu:latest

RUN apt-get -y update 

RUN apt-get -y upgrade 

RUN apt-get install -y build-essential 

RUN apt-get -y install git

RUN cd ~

RUN git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021

Uruchomiłam kontener za pomocą: sudo docker run -it esz-docker
