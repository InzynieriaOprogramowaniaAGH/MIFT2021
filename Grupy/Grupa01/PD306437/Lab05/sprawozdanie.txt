Zadanie 2 
Utworzyłem kontener dockerowy według instrukcji https://www.jenkins.io/doc/book/installing/docker/
Aby ego dokonać musiałem użyć tego polecenia:
sudo docker network create jenkins

Uruchomić kontener dockerowy za pomocą tego polecenia
sudo docker run \
  --name jenkins-docker \
  --rm \
  --detach \
  --privileged \
  --network jenkins \
  --network-alias docker \
  --env DOCKER_TLS_CERTDIR=/certs \
  --volume jenkins-docker-certs:/certs/client \
  --volume jenkins-data:/var/jenkins_home \
  --publish 2376:2376 \
  docker:dind \
  --storage-driver overlay2
  
Stworzyć kolejny plik dockerowy ze zmodyfikowanym obrazem jenkonsowym:
FROM jenkins/jenkins:2.277.1-lts-jdk11
USER root
RUN apt-get update && apt-get install -y apt-transport-https \
       ca-certificates curl gnupg2 \
       software-properties-common
RUN curl -fsSL https://download.docker.com/linux/debian/gpg | apt-key add -
RUN apt-key fingerprint 0EBFCD88
RUN add-apt-repository \
       "deb [arch=amd64] https://download.docker.com/linux/debian \
       $(lsb_release -cs) stable"
RUN apt-get update && apt-get install -y docker-ce-cli
USER jenkins
RUN jenkins-plugin-cli --plugins "blueocean:1.24.5 docker-workflow:1.26"

Zbudować obraz:
docker build -t myjenkins-blueocean:1.1 .

Uruchomić:
sudo docker run \
  --name jenkins-blueocean \
  --rm \
  --detach \
  --network jenkins \
  --env DOCKER_HOST=tcp://docker:2376 \
  --env DOCKER_CERT_PATH=/certs/client \
  --env DOCKER_TLS_VERIFY=1 \
  --publish 8080:8080 \
  --publish 50000:50000 \
  --volume jenkins-data:/var/jenkins_home \
  --volume jenkins-docker-certs:/certs/client:ro \
  myjenkins-blueocean:1.1
  
  Kolejnym krokiem było dokonanie ustawień jenkinsa pod adresem http://localhost:8080 

Zadanie 3
Aby wrzucić obraz na dockerhub utworzyłem repozytorium na dockerhubie poprzez stronę internetową.
Nazwałem je myjenkins-blueocean

Następnie otagowałem jenkinsowy obraz poleceniem:
sudo docker tag 77d33abb1e1b pduda98/myjenkins-blueocean

Następnie spushowałem obraz poleceniem
sudo docker push pduda98/myjenkins-blueocean
