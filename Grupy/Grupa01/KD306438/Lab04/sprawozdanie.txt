1. Utworzenie z brancha grupowego swojego brancha osobistego na te zajecia: 'git checkout -b Grupa01-KD306438_Lab04'

1.1 Opublikowanie obrazu z poprzednich zajec w serwisie Docker Hub: 
	-nadanie zbudowanemu obrazowi tagu takiego samego jak repo, do ktorego bedziemy pushowac: 'sudo docker tag a7c66ceffe6d karen2035/devops'
	-zalogowanie sie na konto 'sudo docker login'
	-opublikowanie obrazu 'sudo docker push karen2035/devops'


2. Znalezienie w internecie komunikatora opensource messeging app - Zulip

3. Build agent: utworzenie nowego obrazu kontenera na podstawie Dockerfile'a realizujacego budowanie komunikatora: 
	-Dockerfile:
		FROM node:14.16.0-buster
		# instalacja gita
		RUN apt-get update -y
		RUN apt-get upgrade -y
		RUN apt-get install --assume-yes git
		RUN apt-get install --assume-yes g++
		RUN git clone https://github.com/zulip/zulip-desktop.git
		WORKDIR ./zulip-desktop
		RUN npm install
		CMD ["npm", "start"]
	 -budowa 'sudo docker build -t im1 . '
	 
4. Test agent: utworzenie nowego obrazu kontenera na podstawie Dockerfile'a realizujacego testowanie zbudowanego wczesniej komunikatora: 
	-Dockerfile:
		FROM im1
		WORKDIR ./zulip-desktop
		RUN npm test
	 -budowa 'sudo docker build -t im1t . -f Dockerfile_test'
