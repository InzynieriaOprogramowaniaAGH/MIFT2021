0. Zapoznano się z diagramem zaprezentowanym na zajeciach.
1. Zapoznao się z oprogramowaniem Jenkins.
2. Utworzono konetener Jenkinsowy zgodnie z instrukcją podaną na stronie: https://www.jenkins.io/doc/book/installing/docker/
3. Wrzucono obraz na stronę dockerhub.com.
4. Uruchomiono kontener Jenkinsa polecenem:
docker run   --name myjenkins   --rm   --detach   --network jenkins   --env DOCKER_HOST=tcp://docker:2376   --env DOCKER_CERT_PATH=/certs/client   --env DOCKER_TLS_VERIFY=1   --publish 8080:8080   --publish 50000:50000   --volume jenkins-data:/var/jenkins_home   --volume jenkins-docker-certs:/certs/client:ro lbielak/myjenkins:latest

4.1 Uruchomienie kontenera jenkins-docker poleceniem:
docker run --name jenkins-docker --rm --detach \
  --privileged --network jenkins --network-alias docker \
  --env DOCKER_TLS_CERTDIR=/certs \
  --volume jenkins-docker-certs:/certs/client \
  --volume jenkins-data:/var/jenkins_home \
  --publish 2376:2376 --publish 3000:3000 docker:dind --storage-driver overlay2
  
  
4.2. W panelu utworzono nowy projekt (freestyle project) i jako repozytorium dodano https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021. Następnie w sekcji "budowanie" uruchomiono powłokę i wpisano następujące komendy:


cd MIFT2021/Grupy/Grupa01/LB299663/Lab04
