0. Zapoznano się z diagramem omawianym na zajęciach.

1. Zapoznano się z oprogramowaniem Jenkins: https://www.jenkins.io/.

2. Stwórzono kontener dockerowy z Jenkinsem posługując się instrukcją https://www.jenkins.io/doc/book/installing/docker/

sudo docker network create jenkins
sudo docker run --name jenkins-docker --rm --detach \
   --privileged --network jenkins --network-alias docker \
   --env DOCKER_TLS_CERTDIR=/certs \
   --volume jenkins-docker-certs:/certs/client \
   --volume jenkins-data:/var/jenkins_home \
   --publish 2376:2376 docker:dind --storage-driver overlay2

Stworzono plik Dockerfile zawarty w folderze Lab05.

Zbudowano obraz: sudo docker build -t myjenkins-blueocean:1.1 .

Uruchomiono obraz: sudo docker run --name jenkins-blueocean --rm --detach \
   --network jenkins --env DOCKER_HOST=tcp://docker:2376 \
   --env DOCKER_CERT_PATH=/certs/client --env DOCKER_TLS_VERIFY=1 \
   --publish 8080:8080 --publish 50000:50000 \
   --volume jenkins-data:/var/jenkins_home \
   --volume jenkins-docker-certs:/certs/client:ro \
   myjenkins-blueocean:1.1
   
3. Wrzucono obraz swojego Jenkinsa na DockerHub'a:
sudo docker tag 8669704006c3 nataliaczachor14/lab05
sudo docker push nataliaczachor14/lab05
