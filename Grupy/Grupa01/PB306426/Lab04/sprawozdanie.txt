1.	Stworzono nowego brancha na zajęcia
		git checkout -b Grupa01-PB306426_Lab04
	Opublikowano obraz z poprzednich zajęci w dockerhubie
	    Commit:
		sudo docker commit 6bea9da706dd pazindor/lab_04:lab_03
	    Push: 
		sudo docker push pazindor/lab_04:lab_03


2. 	Wybrano jako komunikator Wire

3.	Utworzono Dockerfile-build który buduje aplikacje wire-desktop i uruchominie obrazu
		sudo docker build -t wire -f Dockerfile-build ./
		sudo docker run -t wire

4.	Utworzono Dockerfile-test który wykonuje testy
		sudo docker build -t test -f Dockerfile-test ./
		
5.	Instalacja docker-compose, stworznie pliku docker-compose.yml oraz uruchomienie go
	 	sudo curl -L "https://github.com/docker/compose/releases/download/1.29.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
		sudo chmod +x /usr/local/bin/docker-compose
		sudo docker-compose up
