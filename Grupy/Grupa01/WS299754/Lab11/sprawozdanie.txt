2. - Instalacja minikube:
	curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
	sudo install minikube-linux-amd64 /usr/local/bin/minikube
- Uruchmienie klastra
	minikube start
- Instalacja kubectl
	minikube kubectl -- get po -A
3. - Uruchmienie Minikube Dashboard
	minikube dashboar
- Funkcje Minikube Dashboard:
	- wdrażanie konteneryzowanych aplikacji
	- troubleshoting aplikacji
	- zarządzanie zasobami klastra
	- przegląd aplikacji uruchomionych na klastrze
	- tworzenie Cron Jobs, Daemon Sets, Deployments, Jobs, Pods, Replica Sets, Replication Controllers, Stateful Sets
4.  - Utworzenie deploymentu komunikator
	minikube kubectl -- create deployment komunikator --image=wsmieja/node-chat-app
- Przypisanie portu
	minikube kubectl -- expose deployment komunikator --type=NodePort --port=3000
- Uruchomienie usługi
	minikube service komunikator
