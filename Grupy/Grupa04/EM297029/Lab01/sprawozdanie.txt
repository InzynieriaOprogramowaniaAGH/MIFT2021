git clone nazwa repo - klonowanie repozytorium zdalnego - utworzenie repozytorium lokalnego 
git branch nazwa - utworzenie nowego brancha - nazwa: EM297029
git checkout nazwa - przełączenie się na istniejący już branch (mogłam użyć git checkout -b nazwa - żeby automatycznie stworzyło się nowe repo i żebym się przełączyła ale nie zrobiłam tego)

git add . - dodanie zmian do stage
git commit -m nazwa_commita - dodanie commita do naszego repo lokalnego - wszystko co jest w stage zostaje zakomitowane

git push --set--upstream origin nazwa_brancha - w przypadku 1 pusha dla naszego nowego brancha - dodanie naszych zmian do remota i ustawienie na śledzenie tego brancha
git push - kiedy już śledzimy branch - dodanie do remota naszych commitów lokalnych

git checkout master - przełączenie na brach master

git pull - pobranie zależności, których nie mamy lokalnie, a istnieją w remote
git merge nazwa - połączenie branchy. Do tego, w którym obecnie jesteśmy zostaną dołączone zmiany z brancha, którego nazwę wpiszemy
