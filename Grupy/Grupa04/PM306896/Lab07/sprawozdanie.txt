zad 2:
stworzono plik Jenkinsfile majacy na celu wykonywanie testow
w po ich wykonaniu w zaleznosci od wyniku na maila wysylana jest informacja o przebiegu testow:
pipeline {
	agent { 
		docker { image 'node:alpine'}
	}
	stages {
		stage('Test') {
			steps {
				sh 'npm test'
			} 
			
		}
		post {
			success{
				mail to: 'pmroz@student.agh.edu.pl', 
				subject: 'The pipeline ${currentBuild.fullDisplayName}tests succeeded'
				body: 'test succeeded'
			}
			failure{
				mail to: 'pmroz@student.agh.edu.pl', 
				subject: 'The pipeline ${currentBuild.fullDisplayName} tests failed'
				body: '${env.BUILD_URL}'
			}
		}
		
	}
}
