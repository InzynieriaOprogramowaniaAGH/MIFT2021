(
sudo docker tag myjenkins-blueocean:1.1 szymou/jenkins
sudo docker push szymou/jenkins
)

zadanie 4
uruchomiono kontener za pomocą komendy:
docker run \
  --name jenkins-blueocean \
  --rm \
  --detach \
  --network jenkins \
  --env DOCKER_HOST=tcp://docker:2376 \
  --env DOCKER_CERT_PATH=/certs/client \
  --env DOCKER_TLS_VERIFY=1 \
  --publish 8080:8080 \
  --publish 50000:50000 \
  --volume jenkins-data:/var/jenkins_home \
  --volume jenkins-docker-certs:/certs/client:ro \
  myjenkins-blueocean:1.1 
  
Uruchomiono kontener a następnie znaleziono hasło  w folderze /var/jenkins_home/secrets/initialAdminPassword.
Uruchomiono localhost:8080 w przegladarce oraz podano wczesniej znalezione hasło
