Sprawozdanie nr 3

1. Stworzenie brancha osobistego.
-> git branch -b checkout Grupa04-JM299151_Lab03

2. Dodanie git hooka do folderu.
-> cp commit-msg ../../Grupy/Grupa04/JM299151/Lab03

3. Zainstalowanie dockera.
 $ sudo apt-get update 
 $ sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg \
    lsb-release
 $ curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
 $ sudo apt-get update
 $ sudo apt-get install docker-ce docker-ce-cli containerd.io
3.1 Sprawdzenie wersji zainstalowanego dockera 
 $sudo docker version

4.Zarejestrowanie konta w Docker HUB.

5.Stworzenie pliku Dockerfile
-> touch Dockerfile
-> cat Dockerfile:
	FROM ubuntu:latest
	RUN apt-get -y update 
	RUN apt-get -y upgrade 
	RUN apt-get install -y build-essential 
	RUN apt-get -y install git
	RUN cd ~
	RUN git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021 


6.Uruchomienie kontenera, oraz ściągnięcie go do repozytorium.
-> sudo docker build -t jm-docker .
-> sudo docker run -t jm-docker
