1. Zainstalowałam klienta GIT.

2. Po uruchomieniu konsoli sklonowałam repozytorium za pomoca polecenia
   git clone 'adres_strony'. Narysowałam graf zależności pomiędzy dwoma 
   repozytoriami.

3. Zapoznałam się z dokumentem.

4. Utworzyłam swój branch za pomocą komendy git checkout -b KL306676, 
   następnie za pomocą komendy mkdir utworzyłam folder o tej samej 
   nazwie oraz o nazwie Lab01. Użyłam komend:
   git add . - aby dodać wszystkie pliki znajdujące się w Lab01
   git commit -m "commit"
 
5. Utworzyłam niniejszy plik za pomocą polecenia touch. Użyłam polecenia
   git push aby dodać zawartość folderu na remota

6. Do folderu Lab01 dodałam screeny z wykorzystanymi komendami. Następnie 
   użyłam komendy git add .,a potem git commit. Przełączyłam się na master 
   a następnie zmergowałam swojego brancha.

7. Wykonałam merge do głównego brancha za pomocą komendy
   git merge KL306676
   Uprzednio przełączyłam się na głównego brancha. Potem użyłam następujących
   komend: git pull oraz git push
